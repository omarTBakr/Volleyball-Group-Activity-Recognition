\section{Introduction}

    \taustart{V}olleyball Group Activity Detection dataset was originally a dataset held to classify the action of a group of players in a volleyball match ,
    the data was introduced as a part of this paper \href{Paper in CVPR}{https://www.cs.sfu.ca/~mori/research/papers/ibrahim-cvpr16.pdf}
    you can download the data set from \href{Dataset Download}{https://github.com/mostafa-saad/deep-activity-rec} 
    and you can find it on kaggle here \href{Kaggle Download}{https://www.kaggle.com/datasets/ahmedmohamed365/volleyball}

\section{Dataset Summary}
  
we have 8 action classes across the whole Image :
  \begin{itemize}
    \item Moving
    \item Standing
    \item Blocking
    \item Waiting
    \item Setting
    \item Digging
    \item Falling
    \item Jumping
  \end{itemize}
  
  firstly  Standing class is the most frequent frame in the dataset compared to other classes , resulting in huge imbalance in the dataset as seen in Fig \ref{fig:figures/tracking_classes_dist}
  \begin{figure}
    	     \centering
    	     \includegraphics[width=0.95\linewidth]{figures/action_classes_distribution.png}
    	     \caption{distribution of action classes across different videos }
    	     \label{fig:action_classes_distribution}
    	 \end{figure}
    
   ,therefore one way is to downsample the standing class.
 
 
	
\section{Abstract}
 I have come up with what I think are some decent features, as follows:
   \begin{enumerate}
    \item Time-related features: \texttt{is\_weekend}, \texttt{is\_rushhour}, features directly extracted from the datetime object of the pickup, and also some cyclic time features (a fancy name for applying $\sin$ and $\cos$ functions for each feature).
    
    \end{enumerate}

        \begin{itemize}
